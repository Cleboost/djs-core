## Basic

### Inline Code

Use inline code to display code snippets within text paragraphs. It's ideal for referencing code elements directly in sentences.

`` `inline code` ``

### Code Blocks

Use code blocks to display multi-line code snippets with syntax highlighting. Code blocks are essential for presenting code examples clearly.

```ts
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```

When writing a code-block, you can specify a filename that will be displayed on top of the code block.

```ts [nuxt.config.ts]
export default defineNuxtConfig({
  modules: ['@nuxt/ui']
})
```

Every code-block has a built-in copy button that will copy the code to your clipboard.

## Advanced

### CodeGroup

Group code blocks in tabs using `<CodeGroup>`. This is perfect for showing code examples in multiple languages or package managers.

<CodeGroup>
```bash pnpm
pnpm add @djs-core/runtime
```

```bash npm
npm install @djs-core/runtime
```

```bash bun
bun add @djs-core/runtime
```
</CodeGroup>
